# brockhoff-cloud

Claude Code Plugins by kbrockhoff

## Installation

Add the marketplace to Claude Code:

```
/plugin marketplace add kbrockhoff/brockhoff-claude-marketplace
```

Install a specific plugin:

```
/plugin install example-plugin@brockhoff-cloud
```

## Plugins

- **example-plugin**: Example plugin demonstrating command structure

## Development

Run the linter to validate plugin structure:

```bash
make lint
```

Update plugin documentation and website:

```bash
make update
```
